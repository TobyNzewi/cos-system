## How to use it

The folder is wired up to a scheduled task at `~/Documents/Claude/Scheduled/weekly-chief-of-staff/SKILL.md` that runs every <DAY> at <TIME>.

Manual use (anytime):

- Open Cowork → start a chat in this project. Claude reads `_system-context.md` automatically.
- Or paste the contents of `_system-context.md` into a Claude.ai chat to ground a fresh session.

## Maintenance

- **Update `_system-context.md` every few weeks.** It goes stale. The "active tracks" and "files" sections drift first.
- **Update `_<track>.md` every session on that track.** The "where I left off" line is the most important line in the doc — next session reads it first.
- **Don't tinker with the system between sessions.** Use it for two weeks before iterating.
